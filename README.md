# PRODIGY_WD_01
A responsive landing page featuring a fixed, interactive navigation menu with scroll and hover effects. Developed using HTML, CSS, and JavaScript as part of Prodigy InfoTech Web Development Internship (Task-01).

This project is a **responsive landing page** developed as **Task-01** of the **Prodigy InfoTech Web Development Internship**.  
It features a **fixed and interactive navigation menu** that changes its style on scroll and hover, ensuring a clean and user-friendly experience across all devices.
